HOW IT WORKS
Type a name (e.g. smartscreen) and the target IP:port (e.g. http://10.0.0.5:8080). Every device on your network can then open http://smartscreen.lan in a browser — no port number needed. DNS is configured automatically.
SCHEDULE RULES
Label Device / IP Days Time Window Status
No schedule rules yet — add a rule to block a device during specific hours.
HOW SCHEDULING WORKS
Schedule rules block all DNS queries from a device during the specified window — effectively cutting off internet access. Use * as the device IP to apply a rule to all devices. Times use 24-hour format. Overnight ranges (e.g. 22:00 → 07:00) are supported. Rules take effect within 30 seconds of being saved.

SERVICES & FEATURES
YouTube Transparent Proxy
Redirects YouTube DNS to this server. nginx intercepts HTTPS, blocks ad requests, and forwards video traffic to real YouTube. Requires CA certificate on devices.
Captive Portal
Shows a certificate install page to new devices. OS captive portal detection domains are resolved to this server.
NTP Server port 123/UDP
Serves accurate time to all LAN devices. Syncs from pool.ntp.org, time.cloudflare.com, and time.google.com. Point your router or devices' NTP setting to this server's IP.
Changes apply automatically — no save needed. DNS hot-reloads within 30 seconds.
